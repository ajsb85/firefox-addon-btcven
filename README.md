firefox-addon-btcven
====================

This very tiny (15KB) add-on will add a "bitcoin average" menu item to the "File" menu, optional toolbar button

BPI Price from CoinDesk.

http://bitcoinvenezuela.com/api/btcven.json

currency money:
VEF = Venezuela 
ARS = Argentina 
USD = USA 
EUR = Europa
